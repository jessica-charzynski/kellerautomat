Aufgabenstellung

Erzeugen Sie mit Hilfe der Grammatik aus Aufgabe P2 einen zufälligen arithmetische Ausdruck. Es sollt auch möglich sein einen [wahrscheinlich] fehlerhaften Ausdruck zu erzeugen. Dazu löschen Sie einfach ein beliebiges Zeichen aus dem Ausdruck. Prüfen Sie den erzeugten Ausdruck mittels eines Kellerautomaten.

Technische Dokumentation

Das User Interface habe ich mithilfe der Entwicklungsanwendung Bootstrap Studio gestaltet. 
Mithilfe dem Framework IntroJS habe ich eine kleine Hilfe-Tour durch meine Anwendung eingebaut.

Zunächst wird ein zu überprüfender Ausdruck erstellt. Das kann durch eine freie Texteingabe oder über die zwei Buttons erfolgen. Im Anschluss wird die Animation im Kellerautomat durch einen Klick auf dem Play-Button gestartet.
Für die Gestaltung der Schritte habe ich die JavaScript-Bibliothek jQuery verwendet.

Meine Anwendung ruft den zu überprüfenden Ausdruck auf und wandelt ihn in ein Array aus den einzelnen Zeichen um. Anschließend wird Zeichen für Zeichen überprüft.

Wenn eine ungültige Eingabe vorliegt, wird die Animation beendet und die Ausgabe „Ausdruck ist ungültig“ erfolgt.

Liegt hingegen eine gültige Eingabe vor, kann eine tiefer gelegene Ästelung in der if-else-Verzweigung bearbeitet werden. Dabei wird das nächste Zeichen überprüft. 

Dies erfolgt solange bis
a) ein ungültiges Zeichen überprüft wird und somit die Animation mit einer entsprechenden Rückmeldung beendet wird.
b) alle Zeichen überprüft wurden. Ist dabei der Endzustand erreicht worden und liegt ein leerer Keller vor, handelt es sich um eine gültigen Ausdruck der Grammatik und die Rückmeldung „Ausdruck ist gültig“ erscheint. Wurde jedoch der Endzustand nicht erreicht und/oder befinden sich noch Zeichen im Keller, dann handelt es sich um einen ungültigen Ausdruck und eine entsprechende Rückmeldung wird ausgegeben.

Nach Abspielen der Animation kann mit dem "Anwendung resetten"-Button die Seite mithilfe der Funktion "reloadThePage()" in ihren Ausgangszustand zurückversetzt werden.
